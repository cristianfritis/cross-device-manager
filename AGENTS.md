# Repository Agent Instructions

## Native UI Design

Before modifying `gui/**`, `tui/**`, or user-visible behavior in `app/**`, read
`docs/DESIGN.md`. Treat its **must** statements as project requirements. Preserve
GUI/TUI semantic parity while using native Qt Widgets and FTXUI patterns.

The design guide is the single source of truth. Do not duplicate or replace its
rules with provider-specific visual defaults.
